# Resistance Training: Rings Workout Base Plan
## Working Hypothesis: Rings allow for adjusting difficulty more easily as leg support or changing the angle of the exercise will increase or decrease the difficulty

## Intended Schedule

W: Pull Workout
R: Rest
F: Full Body Workout
S: Rest
S: Rest

### The Push Workout
A.    Ring Support Hold - 2 x 30s
B.    Ring Push Up - 4 x 6-10r
C.    Ring Assisted Dip - 3 x 8-12r
D.    OPTIONS (Pick One)
    D1. Ring Tricep Extension - 2 x 10-12r
    D2. Ring Face Pull - 2 x 10-12r 


### The Pull Workout
A.  Ring Chin Ups / Assisted L Chin Up - 4 x 6-10r
B.  Ring Bodyweight Row - 4 x 8-12r
C.  Ring Bicep Curl - 3 x 8-12r
D.  False Grip Hang - 2 x 30s
