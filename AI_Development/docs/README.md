# AI Development Documentation

This directory contains the public-facing documentation for the AI Development project. These are the curated guides and educational materials shared on the portfolio.

## Contents

- **curriculum.md** - A comprehensive, progressive four-stage curriculum for developing professional competency with AI systems. Progresses from beginner exploration through professional mastery.

## Using This Documentation

This curriculum represents a structured approach to AI development suitable for both personal learning and as a reference for others looking to deepen their AI proficiency.

For supporting reference materials (personas, task tracking, etc.), see the `reference/` directory.

## Philosophy

This curriculum is based on the principle that effective AI integration requires moving beyond casual tool usage to systematic skill development with measurable progression milestones.
