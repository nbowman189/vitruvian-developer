## Visual Principles

1. **Visual Hierarchy**
   - Hero > Sidebar > Main Content
   - Important info first, details later
   - Color and size create emphasis

2. **Whitespace**
   - 3rem padding around sections
   - 2rem margins between sections
   - Breathing room for readability

3. **Color Usage**
   - Discipline colors communicate context
   - Accent color (orange) highlights CTAs
   - Neutral grays for supporting elements

4. **Typography**
   - System fonts for performance
   - Clear hierarchy with sizing
   - Generous line-height for comfort
   - High contrast for accessibility

5. **Responsive Design**
   - Sidebar stacks on tablet
   - Single column on mobile
   - Touch-friendly buttons (48px minimum)
   - Text scales appropriately

## Animation & Transitions

```
Hover Effects (0.2-0.3s ease):
```
